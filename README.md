- EXCAVATOR PRODUCTIVITY APP
Прототип программной системы для расчета производительности карьерного экскаватора.

- ВОЗМОЖНОСТИ СИСТЕМЫ
расчет часовой производительности;
расчет годовой производительности;
учет технического обслуживания;
учет аварийных простоев;
расчет коэффициента технической готовности;
проверка выполнения производственного плана;
формирование рекомендаций.

- ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ
1) Python 3.13
2) FastAPI
3) HTML/CSS/JavaScript
4) Docker
5) Pytest

- СТРУКТУРА ПРОЕКТА
app/
├── main.py
├── calculator.py
├── schemas.py
├── templates/
│   └── index.html
└── static/
    ├── style.css

- ЗАПУСК БЕЗ DOCKER
1) УСТАНОВКА ЗАВИСИМОСТЕЙ
pip install -r requirements.txt
2) ЗАПУСК ПРИЛОЖЕНИЯ
uvicorn app.main:app --reload
ОТКРЫТЬ В БРАУЗЕРЕ
http://127.0.0.1:8000

- ЗАПУСК ЧЕРЕЗ DOCKER
СБОРКА И ЗАПУСК
docker compose up --build

- ОТКРЫТЬ В БРАУЗЕРЕ
http://localhost:8000

- ТЕСТИРОВАНИЕ
Запуск тестов:
